# AM Typography

Source: `Guidelines/DesignGuides/TypographyScale` (Storybook).

- **Font family**: `"Roboto", sans-serif` — load via Google Fonts in standalone mockups.
- **Weights in use**: 400 (regular), 500 (medium), 700 (bold).
- **Size scale (px)**: 12 (label), 13 (helper), 14 (body), 16 (section), 20 (page), 24 (hero). Don't use sizes outside this scale.

## Application text patterns

These are the baseline for the most common text in the product. Note: page and section headers are **blue**, not black — this is one of the most recognizable traits of the AM look.

| Pattern | Token | Style | Usage | Example |
|---|---|---|---|---|
| Page Header | h1 / 20–24px / 700 | 24px, 700, `#4C8BA5`, line-height 1.2 | Page-level titles, primary screen headings | "Bearbeitungsdetails" |
| Section Header | h2–h3 / 16–20px / 700 | 20px, 700, `#4C8BA5`, line-height 1.2 | Card titles, sections, grouped content headings | "Die wichtigsten Daten auf einen Blick" |
| Body Text | body / 14px / 400 | 14px, 400, `#2B2B2B`, line-height 20px | Default content, values, paragraphs, table text | "Der Klient hat Interesse an einem Rückruf…" |
| Label Text | label / 14px / 700 | 14px, 700, `#2B2B2B`, line-height 1.3 | Form labels, short field titles | "E-Mail-Adresse" |
| Helper Text | helper / 13px / 400 | 13px, 400, `#707070`, line-height 1.4 | Supporting form guidance, quiet metadata below fields | "Bitte geben Sie eine gültige E-Mail-Adresse ein." |
| Info Text | info / 14px / 400–700 | 14px, 400, `#4C8BA5`, line-height 1.4 | Informational notices, hints that stand out without feeling like an error | "Hinweis: Änderungen an Ihrer Buchung werden nicht gespeichert." |
| Success Text | success / 14px / 700 | 14px, 700, `#98C44C`, line-height 1.3 | Positive values, success labels, supportive highlights | "Gewonnen" |
| Error Text | error / 14px / 400–700 | 14px, 400, `#E74C3C`, line-height 1.4 | Validation feedback, error messaging, destructive text | "Dieses Feld ist erforderlich." |
